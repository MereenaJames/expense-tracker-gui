Expense Tracker - Upgraded GUI

A **Python Tkinter** based graphical expense tracker that helps you manage and visualize your daily expenses efficiently. The application allows you to add, view, delete, and export expenses, as well as visualize spending trends through charts.

---

 Features

- Add expenses with **description, amount, and category**.
- Set a **monthly budget** and track remaining budget.
- View expenses in a **table format**.
- **Delete selected expense entries**.
- Generate **category-wise pie chart**.
- View **daily expense trend** in a line chart.
- **Export expenses to Excel** for further analysis.
- Supports categories: `Food`, `Travel`, `Shopping`, `Other`.
